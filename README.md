# Система управления студентами

Программа для управления информацией о студентах, реализованная на языке C11 с использованием объектно-ориентированного подхода.

## Описание

Программа предоставляет функционал для:
- Создания студентов с персональными данными
- Управления курсами студентов
- Оценки успеваемости на основе среднего балла
- Классификации студентов по курсам обучения
- Работы с несколькими студентами одновременно

## Требования

- Компилятор C с поддержкой стандарта C11 (GCC, Clang)
- Операционная система: Linux, macOS, Windows (с MinGW)
- Минимальный объем оперативной памяти: 128 MB

## Компиляция и запуск

# Сборка программы
make

# Запуск программы
./student_demo

make clean
